# Customer_Behavior_Analysis
Data Analytics project displaying customer behavior using Python, SQL and Power BI

## 🔍 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data and ending with actionable insights and visual storytelling. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based querying, dashboard creation, and reporting.
The goal is to showcase practical skills in data analysis, visualization, and communication using industry-standard tools.


 

## 📁 Dataset
 
The dataset used in this project contains structured data relevant to business analysis.

It includes multiple features such as categorical and numerical variables.

Data was sourced from a publicly available dataset

## 🛠️ Tools & Technologies

   -Python (Pandas) – Data processing and EDA

   -PostgreSQL – Data storage and SQL querying

   -Power BI – Interactive dashboard creation

   -Gamma – Presentation (PPT) generation

   -Jupyter Notebook – Development environment

##   ⚙️ Project Steps
   
1. Data Loading

    -Imported dataset using Python (Pandas)
  
    -Performed initial inspection (shape, data types, missing values)

2. Exploratory Data Analysis (EDA)

    -Analyzed distributions and relationships between variables
   
    -Identified trends, patterns, and potential anomalies
   
    -Visualized key insights using plots and charts
   
3. Data Cleaning
   
    -Handled missing values
   
    -Removed duplicates and inconsistencies
   
    -Standardized formats and corrected data types

4. SQL Analysis (PostgreSQL)
   
    -Loaded cleaned data into PostgreSQL
   
    -Wrote SQL queries for deeper analysis

    -Extracted insights for reporting
   
5. Dashboard Creation (Power BI)

    -Built an interactive dashboard

    -Enabled easy exploration of insights
   
6. Reporting & Presentation
   
    -Created a structured analytical report
   
    -Designed a presentation using Gamma
   
    -Highlighted key findings and recommendation
